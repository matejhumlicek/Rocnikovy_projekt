# Remote controlled BB-8 Droid

 Ročníkový projekt
  
 Cílem mého projektu je vyrobit pojizdného autentického droida ze série filmu Star Wars -- BB-8.  
 Droid by měl být ovladatelný přes mobilní aplikaci.
